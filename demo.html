Добро пожаловать на Демонстрационную страницу JCat Online Player.

Ниже будет приведено описание установки, подключения и настройки плеера.

Установка плеера

Для начала вам нужно:

1. Загрузить скрипт на сайт (папки css, font и js).

2. Загрузить файл info.xsl из папки to icecast2 web folder в папку веб части сарвера Icecast.
По умолчанию /usr/share/icecast2/web для Debian (можно посмотреть в конфиге Icecast-a).

Подключение плеера

Так как плеер работает на JQuery то нужно обязательно его подключить. Вы можете использовать JQuery, который шёл вместе с плеером, и подключить его прописав в коде страницы:

<script type="text/javascript" src="jquery.min.js"></script>
либо не подключать, если он у вас уже подключен на сайте.

Также нужно подключить файлы плеера:

<link rel="stylesheet" type="text/css" href="css/jcplayer.min.css">
Если у вас главный поток вещания называется live, а фоновый nonstop, то используйте:
<script type="text/javascript" src="js/jcplayer.min.js"></script>
Иначе:
<script type="text/javascript" src="js/jcplayer.js"></script>
Предварительно поправив в этом файле в строке 90, 95 и 97 названия потоков вещания.
Например: (Название главного потока play, фонового shadow) 
...
  if (d.play.title == '') {einfo = true} else einfo = false;//First Mount (Main)
...
    $('#jcp-track').html(d.shadow.title);//Second Mount (Fallback)
  else
    $('#jcp-track').html(d.play.title);//First Mount (Main)
...

Настройка плеера

Настройка состоит из нескольких пунктов:

1. Добавить после строк подключения плеера следующее: 

<script type="text/javascript">
    src = 'http://127.0.0.1:8000/live'; //Основной поток вещания
    info = 'http://127.0.0.1:8000/info.xsl'; //Информационный файл
    tupd = 10; //Время обновление информации (в секундах)
</script>
Где пременная src - это ссылка на главный поток,
пременная info - это ссылка на файл info.xsl на сервере вещания Icecast,
а пременная tupd - это интервал времени обновления названия играющего трека в секундах.

2. Добавить в любое место сайта (для правильного отображения плеера, рекомендуется перед тегом </body>):

<div id="jcp-player"></div>

3. Добавить на страницу, на которой выводите плеер: 

<style>
body {padding-bottom:61px;}
@media (max-width: 767px) {body {padding-bottom:124px;}}
#jcp-player {display:block}
</style>

4. Также для правильной работы плеера на мобильных устройствах настоятельно рекомендуется использовать meta viewport:

<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no">
В таком случае у вас должна быть мобильная или адаптивная вёрстка.
